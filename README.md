## Quickstart

Run:

```bash
cd game.code
python3 -m http.server 5500
```

Open: http://localhost:5500/index.html

Test:
- Play all 11 decisions.
- Check interceptors appear after Q4 and Q7.
- Confirm ending + restart work.
